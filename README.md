## Project Overview
LeafLens is developed to **assist farmers and gardeners** in identifying plant leaf diseases at an early stage. Traditionally, manual inspection is time-consuming and requires expertise. LeafLens automates this process using a **CNN-based classifier**, allowing rapid and accurate detection of multiple leaf diseases. ğŸŒ¿ğŸ§
